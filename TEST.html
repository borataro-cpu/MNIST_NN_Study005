Python For文 動作可視化

for d in test_data: で値が変数に入るタイミングを確認しましょう。

For文は、リストに入った要素の数だけ処理を繰り返します。

今回は test_data = [0.4, 0.9, 1.5, 0.7] と要素が 4つ あるので、合計 4回 繰り返します。

自動再生
1行ずつ進める
最初から
1. 待機中のリスト (test_data)
残り: 4
2. ループ変数: d
# いま代入されている値 None
3. Python Code
test_data = [0.4, 0.9, 1.5, 0.7] # 要素が4つ
print("--- データ取り出しスタート ---")
for d in test_data: # ★ここで代入ログが出る
print(f"取り出した値: {d}")
進捗ログ
「次へ」を押して開始

イメージの解説

for d in test_data: が実行されるたびに、リストの先頭から値が1つ d に入り、リストからは消えていきます。 リストの要素が4つなら、この「取り出し」と「プリント」のセットを合計4回繰り返します。
